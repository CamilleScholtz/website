**Music Player Daemon**
[0.21.19](/download/mpd/0.21/mpd-0.21.19.tar.xz)
has been released.

This release allows overriding configuration settings in includes.
PulseAudio's maximum sample rate is now obeyed, which fixes DSD128
playback.
