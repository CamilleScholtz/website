**Music Player Daemon**
[0.20.4](/download/mpd/0.20/mpd-0.20.4.tar.xz)
has been released.

It fixes a freeze bug in the `nfs` input plugin and a few build
failures (`sndio`/`roar`, GCC 4.9, FreeBSD).
